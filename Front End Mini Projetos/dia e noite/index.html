O JavaScript é usado para alternar entre os temas de dia e noite.
Quando o botão é clicado, o código verifica se o corpo da página tem a
classe "light-theme". Se sim, ele adiciona a classe "dark-theme" e
remove a classe "light-theme", e vice-versa. Isso altera a cor de fundo e a
cor do texto da página, bem como o estilo do cabeçalho e do rodapé.

No código JavaScript que você me mostrou, se eu invertesse
a ordem do "light-theme" e "dark-theme" iria funcionar igualmente?

Sim, você pode inverter a ordem das classes "light-theme" e "dark-theme" no CSS e no JavaScript e ainda funcionará da mesma maneira. A ordem das classes no CSS não importa, pois o navegador irá aplicar o estilo de acordo com a especificidade do seletor e a ordem em que os estilos foram definidos. No JavaScript, a ordem das classes também não importa, pois a função toggle usada em classList.toggle('light-theme') e classList.toggle('dark-theme') remove a classe que já existe e adiciona a classe que não existe. Então, se a ordem das classes no código JavaScript for invertida, o resultado final ainda será o mesmo.
