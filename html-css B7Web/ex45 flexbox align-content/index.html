align-content: é para quando temos mais de um conteúdo. Pode usar como se fosse o justify-content para linhas caso a linha quebre, ficará justificado conforme definido.

Esta propriedade align-content: não tem efeito em containers flexíveis de linha única (ou seja, aqueles com flex-wrap: nowrap)
